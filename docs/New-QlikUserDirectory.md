# New-QlikUserDirectory

## SYNOPSIS

 [-configured <String>] [-syncOnlyLoggedInUsers <String>] [-syncStatus <String>] [-configuredError <String>]
 [-operationalError <String>] [-settings <Object[]>] [-customProperties <String[]>] [-tags <String[]>]
 [<CommonParameters>]
